# rem 布局

## em 和 rem的基本认知









## 为什么要用 rem

1. 流式布局存在的问题
2. 响应式布局的问题









## rem 适配的原理

使用 rem 为单位, 适配 320, 375 的屏幕

rem的核心原理: 屏幕大, 让html的font-size大, 屏幕小, 让html的font-size小









## rem 布局的测试

```
rem 布局的练习: 
   假设有一个设计图: 640, 
   在设计图上量出来一个 div, 200px * 200px  字体大小 40px
   适配的范围
```
